## ModuleIndexBanner 带跳转链接的说明文字区域

### 引入

```javascript
import { ModuleIndexBanner } from '@/components/ui-module-index-banner'

Vue.component(ModuleIndexBanner.name, ModuleIndexBanner)
```

### API

#### ModuleIndexBanner Props

|属性 | 说明 | 类型 | 默认值 | 备注|
|----|-----|------|------|------|
|title | 一级标题 | String | '' | - |
|subTitle | 文字说明 | String | '' | - |
|link | 跳转链接 | String | '' | - |
